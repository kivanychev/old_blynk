Template ID for {template_name} is: {template_id}.

This ID should be added in Settings.h. Simply change this line

#define BOARD_TEMPLATE_ID "{template_id}" // ID of the Tile Template. Can be found in Tile Template Settings

Template ID is used during device provisioning process and defines which template will be assigned to the device of this particular type.
